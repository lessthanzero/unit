Setup and How-to

Unit was made with a simplicity in mind. So, no usual checkboxes or sliders here. Just launch the App, and you’re good to go!

Unit comes with no warranty whatsoever, it is an experiment and shouldn’t be used to make complex financial decisions.

Unit uses open-source Money.js library to do the heavy lifting.

Unit works best as a web app. You can add it to your phone’s home screen:

iOS

  1. Launch Safari
  2. Choose “Add to Home Screen” from sharing menu

Android

  1. Launch Chrome or Firefox
  2. Press menu button, then tap on “Page”
  3. Choose “Add Page Shortcut” from sharing menu

I track all tech issues on Github page. If you want to make Unit better, make a pull request with your suggestion.
